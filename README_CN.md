# Puppet Resume - 后端服务
基于 Node.js、Puppeteer 和 Gemini AI 的专业简历智能重塑与 PDF 生成后端。
[English Version](./README.md)

## 核心能力：简历“自动裁剪”

本项目不仅仅是 PDF 生成器。其核心功能是利用 AI 实现**简历自动裁剪**：根据目标職位的职位描述（JD），自动对用户的原始简历进行内容筛选、重点突出和语言优化，确保每一份生成的简历都能精准契合岗位需求。

## 核心技术亮点

- 🧠 **双模型调度策略**: 智能编排 **Gemini 2.5 Flash** 处理低延迟任务，并自动回溯至 **Gemini 3.0 Pro** 处理复杂推理，兼顾响应速度与改写质量。
- 📸 **多模态感知能力**: 原生支持基于 Gemini Vision 的图片职位详情（JD）提取，允许用户通过截屏即可开启简历生成流程。
- ⚡ **高精度文档渲染**: 采用工业级 **Puppeteer** 渲染内核，驱动动态响应式模板生成像素级、符合行业标准的 PDF 简历方案。
- 🛡️ **生产级稳健性**: 严格遵循工程化标准，内置请求超时管控、自动错误回滚及后台资源自动清理机制，确保服务高可用。
- 💳 **完备金融生命周期**: 深度集成**微信支付**全流程，涵盖安全分发订单、异步回调签名校验及自动化会员权限流转。

## 功能特性

- ✂️ **AI 智能简历裁剪**: 基于职位描述（JD）深度分析，自动重写工作经历和技能描述，实现“一岗一策”。
- 📑 **高质量 PDF 生成**: 基于 Puppeteer 的高保真专业简历文档生成。
- 🤖 **Gemini 引擎动力**: 深度集成 Google Gemini 大模型，提供逻辑清晰、语调自然的简历内容优化。
- 🌍 **全球化支持**: 完美支持中英文简历生成，自动处理日期格式、专业术语等语言差异。

## 技术栈

- **运行时**: Node.js + TypeScript
- **浏览器自动化**: Puppeteer
- **数据库**: MongoDB
- **AI 引擎**: Google Generative AI (Gemini 2.5/3.0)
- **支付集成**: 微信支付 V3 SDK
